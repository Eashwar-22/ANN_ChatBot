# ANN_ChatBot

----

### Requirements:
```shell
flask~=2.0.2
nltk~=3.6.7
numpy~=1.18.5
tensorflow
```

### Virtual Env:
```shell
conda create -n chatbot python=3.8
conda activate chatbot
cd ANN_ChatBot/
pip install -r requirements.txt

```

### Running the app
```shell
export FLASK_APP=app.py
export FLASK_DEBUG=1

flask run app
```
OR
```shell
python app.py
```
